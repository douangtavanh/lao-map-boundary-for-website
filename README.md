# Lao Map Boundary for Website

This repository provides administrative boundary maps at the province, district, and village levels based on 2015 data.

Disclaimer: The CRS of these maps was modified from EPSG:32648 to EPSG:4326 for better compatibility with web mapping. Additionally, some village names have been corrected for typos, and incorrect dcodes allocated to villages have been fixed. 

Data source: National Geographic Department via [k4d.la](https://k4d.la/)
